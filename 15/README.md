# Git assignments

1. Create a folder. 
2. Create some files in the folder.
3. Commit. 
4. Create a branch. 
5. Modify the content of the newly created files. 
6. Commit. 
7. Switch to _main_ branch. 
8. Modify the same files. 
9. Commit. 
10. Try merging branch to _main_ branch. 
11. Try fixing conflicts.
